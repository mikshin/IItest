Промокоды Битрикс24

Внимание! Использовать можно только один из кодов, выбирайте самый нужный!

Промокод дает вашему Битрикс24 дополнительно 5Гб места в облаке сроком на 1 год. Это вдвое больше, чем доступно по-умолчанию на бесплатном тарифе. Применим только для бесплатных тарифов.

Промокод увеличивает максимальное число пользователей на 12, то есть в два раза для бесплатного тарифа Битрикс 24. Применим только для бесплатных тарифов.

Как применить промокод для Битрикс24?

  • В форме регистрации Битрикс24 в самом низу есть кнопка «У меня есть промокод», после нажатия на которую появится строка для указания промокода. Просто укажите в этом поле промокод и ваш бонус уже будет доступен на портале после регистрации.
  • Для тех, кто уже зарегистрировался в Битрикс24, но еще не использовал промокод. Необходимо на вашем портале зайти в раздел Мой тариф – Мой Битрикс24 и нажать на ссылку «Активировать купон». В появившемся поле укажите промокод и нажмите «Активировать».
